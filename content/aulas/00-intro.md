Tente resolver os exercícios antes de consultar os códigos prontos.

## Por que C99?

A linguagem C permite ao programador interagir diretamente com a memória e o hardware, oferecendo uma compreensão profunda de como os programas realmente funcionam. Historicamente, C serviu como base para muitas outras linguagens, como C++, Java e Python, tornando mais fácil aprender e entender essas tecnologias após dominar C. Ao estudar e dominar o padrão C99, o aluno desenvolve uma base sólida em lógica de programação e nos conceitos fundamentais da computação.
